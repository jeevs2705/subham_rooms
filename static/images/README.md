# Room Images

Place your room images in this folder with these exact names:

1. **room-small4.jpg** - Small Room for 4 People
2. **room-small2.jpg** - Mini Room for 2 People  
3. **room-big8.jpg** - Big Room for 8 People
4. **restroom.jpg** - Restroom/Bathroom image

## Image Requirements:
- Format: JPG, JPEG, or PNG
- Recommended size: 800x600 pixels or similar aspect ratio
- Keep file size under 2MB for faster loading

## How to Add Images:
1. Take photos of your rooms and restroom
2. Rename them according to the names above
3. Copy them to this folder: `static/images/`
4. Refresh your website to see the images

If images are not added, placeholder emojis (🛏️ for rooms, 🚿 for restroom) will be shown.
